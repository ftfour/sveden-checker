# ГОСТы по доступности интернет-ресурсов

Эти документы лучше использовать как справочные источники для отдельного модуля «Доступность сайта», а не как ядро проверки раздела `/sveden/`.

## Основной стандарт

- ГОСТ Р 52872-2019 «Интернет-ресурсы и другая информация, представленная в электронно-цифровой форме. Требования доступности для людей с инвалидностью и других лиц с ограничениями жизнедеятельности».

## Дополнительный ориентир

- ГОСТ Р 70176-2022 «Интернет-ресурсы и другая информация, представленная в электронно-цифровой форме. Руководство по применению ГОСТ Р 52872-2019».

## Почему PDF не вложен

Стандарты часто распространяются через официальные каталоги/магазины стандартов, поэтому в архив не добавлены случайные копии PDF из интернета. Для sveden_checker достаточно хранить ссылку/идентификатор стандарта и использовать его только в модуле доступности.
