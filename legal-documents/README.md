# Нормативная база для sveden_checker

Архив собран для разработки локального приложения проверки раздела «Сведения об образовательной организации».

## Что внутри

1. `01_fz-273/` - ссылки на Федеральный закон № 273-ФЗ и статью 29.
2. `02_pp-rf-1802/` - Постановление Правительства РФ № 1802.
3. `03_rosobrnadzor-1493/` - Приказ Рособрнадзора № 1493.
4. `04_rosobrnadzor-1353/` - Приказ Рособрнадзора № 1353 с изменениями к № 1493.
5. `05_rosobrnadzor-955/` - Приказ Рособрнадзора № 955 с проверочными листами.
6. `06_methodical-recommendations/` - Методические рекомендации, версия 9.0.0.
7. `07_official-clarifications/` - официальные разъяснения Рособрнадзора от 11.11.2024.
8. `08_accessibility/` - Постановление Правительства РФ № 102 о требованиях доступности сайта.
9. `09_gost/` - ссылки на ГОСТ по доступности интернет-ресурсов.

## Важно

- Полный PDF 273-ФЗ с official publication portal из этой среды скачать не удалось, поэтому в архив добавлены официальные ссылки и рабочие ссылки на актуальную редакцию/статью 29.
- ГОСТы не вложены PDF-файлами, чтобы не добавлять в архив сомнительные копии стандартов. Для них добавлены ссылки и пометка, что это отдельный справочный источник для модуля доступности.
- Для разработки проверяющего движка используйте `legal-sources.json`: там уже есть id документов, краткие названия и назначение в правилах.
